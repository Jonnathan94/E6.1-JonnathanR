Etiquetas HTML texto

Etiqueta <h1>

La etiqueta <h1> define el nivel del encabezado del documento

Ejemplo

<h1> La historia de Grecia</h1>

Resultado

La historia de Grecia

Etiqueta <del>

La etiqueta <del> se usa para tachar una frase

Ejemplo

La mitología griega es el conjunto de <del>mitos</del> y leyendas

Resultado

La mitología griega es el conjunto de mitos y leyendas

Etiqueta <ins>

La etiqueta <ins> se usa para subrayar una frase

Ejemplo

La mitología griega es el conjunto de mitos y <ins>leyendas</ins>

Resultado

La mitología griega es el conjunto de mitos y leyendas

Etiqueta <br>

La etiqueta <br> se usa para añadir saltos de linias entre párrafos

Ejemplo

La mitología griega es el conjunto <br> de mitos y leyendas

Resultado

La mitología griega es el conjunto
de mitos y leyendas

Etiqueta <dialog>

La etiqueta <dialog> sirve para crear un cuadro de dialogo emergente

Ejemplo

La mitología griega es el conjunto <dialog> de mitos y leyendas</dialog>

Resultado

La mitología griega es el conjunto

de mitos y leyendas

Etiqueta <strong>

La etiqueta <strong> se usa para definir un párrafo importante

Ejemplo

La mitología griega es el conjunto <strong> de mitos y leyendas</strong>

Resultado

La mitología griega es el conjunto de mitos y leyendas

Etiqueta <sub>

La etiqueta <sub>, cuando usas esta etiqueta lo que hace es que el párrafo se muestra en la parte inferior, en letras pequeñas.

Ejemplo

La mitología griega es el conjunto <sub> de mitos y leyendas</sub>

h <sub>2 <sub>o

Resultado

La mitología griega es el conjunto de mitos y leyendas

h 2o

Etiqueta <pre>

La etiqueta <pre> se usa para dar añadir espacio entre un párrafo</pre>

Ejemplo

<pre>La mitología griega es el conjunto de mitos y leyendas</pre>

Resultado

 La mitología griega es el            conjunto de mitos y leyendas

Etiqueta div

<div>

La etiqueta <div> se usa para definir un grupo de contenido o sección de la página, para poder aplicarle diferentes estilos.

Etiqueta cite

Etiqueta <cite>

La etiqueta <cite> se usa para poner un titulo a una imagen, video, musica.

Ejemplo

La Gioconda by Leonardo da Vinci.

Etiqueta des listas

Etiqueta <ul>lista no ordenada

La etiqueta <ul> se usa para crear lista no ordenadas cuando creamos una lista con el tag <ul> usamos también el tag <li> dentro del tag <li> es donde escribimos

Ejemplo

<ul>
<li>Coche </li>
<li>Avión</li>
<li>Barco</li>
</ul>

Resultado

  • Coche
  • Avión
  • Barco

Etiqueta <ol> lista ordenada

La etiqueta <ol>crea listas ordenadas, estas listas puede ser por númerico o alfabético

Ejemplo

<ol>
<li>Coche </li>
<li>Avión</li>
<li>Barco</li>
</ol>

<ol type="A">
<li>Coche </li>
<li>Avión</li>
<li>Barco</li>
</ol>

Resultado

  1. Coche
  2. Avión
  3. Barco
  1. Coche
  2. Avión
  3. Barco

Etiqueta <dl> lista de difiniciones

La etiqueta <dl>se usa para crea una lista con difiniciones , dentro del tag estaran los siguientes tag <dt> <dd>

<dt> dentro de este tag iria el titulo de la difinición.
<dd> dentro de este tag iria la descripción.

Ejemplo

<dl>
<dt>Sol <dt>
<dd>Estrella con luz propia alrededor de la cual gira la Tierra.<dd>
<dt>Estrella<dt>
<dd>Astro o cuerpo celeste que brilla con luz propia en el firmamento.<dd>
</dl>

Resultado

Sol
Estrella con luz propia alrededor de la cual gira la Tierra.
Estrella
Astro o cuerpo celeste que brilla con luz propia en el firmamento.
